# Improve Architecture

Explore the codebase to find architectural improvement opportunities using the improve-codebase-architecture agent. Focus on:

1. **Shallow modules** that should be deepened (classes with little logic)
2. **Tight coupling** between components that should be decoupled
3. **Missing abstractions** that could simplify complex logic
4. **Testability** issues that make the code hard to test

Report specific files and the improvements recommended.
